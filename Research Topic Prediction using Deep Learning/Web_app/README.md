# Frontend & User Interface

The application provides a clean, modern, and interactive user interface built using Streamlit.

The UI is designed to make research paper discovery simple, fast, and user-friendly.

---

## User Interface Features

### Search Section
Users can enter:

- research topics
- paper titles
- keywords
- concepts

into the search box to find semantically similar research papers.

---

### Recommendation Generation
After clicking the **Get Recommendations** button, the system generates the most relevant research papers based on semantic similarity.

---

### Recommendation Cards
Each recommendation card displays:

- Research paper title
- Authors information
- Similarity score
- Research summary preview

This helps users quickly understand the relevance of each paper.

---

### Sidebar Features
The sidebar provides:

- Project information
- Technologies used
- Recommendation controls
- Number of recommendations slider

Users can select how many recommendations they want to display.

---

## Frontend Technologies

The frontend is developed using:

- Streamlit
- HTML/CSS styling
- Python backend integration

---

## UI Design Goals

The interface focuses on:

- Clean layout
- Easy navigation
- Fast interaction
- Readable recommendations
- Interactive semantic search experience

---

## Backend Integration

The frontend is connected with:

- BERT Embeddings
- FAISS Similarity Search
- NLP-based semantic recommendation system

to provide fast and accurate research paper recommendations.
